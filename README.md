# Odin Recipes

This is **Project: Recipes** from The Odin Project Foundations course.

## What I built
A simple multi-page static recipe website with:
- A main `index.html` that lists recipes and links to them
- Individual recipe pages inside a `recipes/` folder

## Skills practiced
- HTML5 document structure
- Headings, lists (ordered + unordered), links, and images
- Organizing files/folders in a small project
- Git + GitHub workflow (commit early & often)
